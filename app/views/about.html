Maps

Esta aplicación permite buscar ubicaciones y crear una lista de ellas, utilizando marcadores.

Mode de uso

Primero se debe buscar una ubicación, utilizando el cuadro de texto superior. Una vez seleccionada una ubicación desde la lista, ésta aparecerá en el mapa, a través de un marcador. Este marcador puede ser movido y ubicado en cualquier posición del mapa. Cuando se tenga la posición donde se quiera ubicar el marcador se puede guardar utilizando el boton ubicado en la parte inferior.

La lista de marcadores de la derecha, muestra los marcadores que ya han sido guardados, aparece la dirección del marcador y dos botones. El primero centra el mapa en la ubicación del marcador, mientras que el segudo elimina el marcador del mapa.

Para agregar una nueva posición se debe borrar el campo de texto para buscar una nueva ubicación, de lo contrario no se agregará ningun marcador al mapa.
